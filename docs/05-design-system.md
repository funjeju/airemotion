# 05 · DESIGN SYSTEM — 디자인 언어

> 이 디자인은 임의 정의가 아니라 **주제(정지 사진이 부드럽게 움직여 영상이 되는 것)**에서
> 끌어낸 방향이다. AI 기본값(크림+세리프+테라코타 / 새까만 배경+형광 / 신문 헤어라인)은
> 의도적으로 피한다.

## 컨셉(Thesis): "Stillness in motion"

정지된 것이 조용히 살아 움직인다. 차분하고 밝고 빛이 도는 **스튜디오** 같은 인상.
시끄럽지 않고 자신감 있게. 화면의 모션 자체가 제품의 모션(드리프트·페이드)을 닮게 한다.

## 시그니처 요소 (이 제품을 기억하게 하는 단 하나)

**가로 "필름스트립 타임라인"** — 클립들이 한 줄로 늘어서고, 각 칸이 켄 번스로 살짝 드리프트하는
미리보기를 보여 준다. 이게 히어로이자 시그니처다. 번호 마커는 **순서가 실제 의미를 가지는
타임라인에서만** 쓴다(장식용 01/02/03 금지).

히어로는 곧 주제다 → 첫 화면에서 **정지 사진이 부드럽게 줌/페이드되는 실제 출력 느낌**을 보여 준다.

## 컬러 토큰

대담함은 **단 한 곳(악센트)**에만 쓴다. 악센트는 "스크린 라이트"를 떠올리게 하는 **인디고**.

### Light
```
--bg          #F4F6F8   /* 데이라이트 페이퍼 (크림 클리셰 회피, 차분한 쿨 화이트) */
--surface     #FFFFFF
--ink         #1A1D21   /* 살짝 따뜻한 차콜 */
--muted       #5B6470
--line        #E4E8EC   /* 헤어라인 아님 — 부드러운 구분선 */
--accent      #5654D4   /* 인디고(스크린 라이트) — 시그니처 */
--accent-weak #ECECFB
--render      #E8B84B   /* 호박색 — '렌더링/활성' 상태에만 아주 드물게 */
```

### Dark (1급 시민, 별도 설계)
```
--bg          #0F1116   /* 순흑 아님 — 푸른기 도는 딥 슬레이트 */
--surface     #181B22
--ink         #E6E8EC
--muted       #8A93A0
--line        #262A33
--accent      #7B79F2   /* 어두운 배경에서 더 떠오르게 밝힌 인디고 */
--accent-weak #1F1F3A
--render      #E8B84B
```

- 악센트는 주요 동작(생성/확정), 활성 클립 테두리, 포커스 링에만.
- `--render`(호박색)는 렌더 진행 등 "지금 작동 중" 신호에만. 남용 금지.

## 타이포그래피

같은 폰트 3종을 역할별로. 데이터(타임코드)는 모노 — **진짜로 시간 값**이라 기능적으로 정당.

```
Display : "Space Grotesk"   — 헤딩·히어로. 성격 있되 절제해서 사용.
Body/UI : "Inter"           — 본문·버튼·라벨. 중립·고가독.
Mono    : "Geist Mono"      — 타임코드·길이(00:04, 0.5s)·수치 입력.
```

스케일(예): Display 40/32/24, Body 16/14, Mono 13. 헤딩은 약간 타이트한 자간,
본문은 편안한 행간. 디스플레이를 "전달 수단"이 아니라 **기억에 남는 요소**로 다룬다.

## 레이아웃

- 넉넉한 여백, 부드러운 그림자, **둥근 모서리(radius ≈ 12px)** — 비전문 사용자에게 친근하게.
  (신문형 zero-radius·헤어라인은 의도적으로 안 씀.)
- 에디터 화면 구조(ASCII):

```
┌───────────────────────────────────────────────┐
│ 로고      프로젝트명        🌐언어  🌗테마  👤   │  상단바
├───────────────────────────────────────────────┤
│  [ 미리보기 캔버스 — 켄 번스 + 크로스페이드 재생 ]│
│                                                 │
├───────────────────────────────────────────────┤
│  필름스트립 타임라인 (시그니처)                  │
│  ① [사진] ② [영상] ③ [사진] …  ＋업로드        │
│        └ 클립 클릭 시 아래 인스펙터 열림         │
├───────────────────────────────────────────────┤
│  인스펙터: 자막 입력 · (펼치면) 색/크기/위치     │
└───────────────────────────────────────────────┘
                         [ 영상 만들기 ]  ← 악센트
```

## 모션 (주제와 일치)

- UI 모션은 제품 모션을 닮게: **부드러운 드리프트·페이드·살짝 떠오르는 호버.**
- 켄 번스 미리보기, 0.4–0.6s 크로스페이드, hover 시 미세한 lift.
- 이징은 등속 금지 — `spring()` 또는 ease-in-out. 산만한 효과 남발 금지(과하면 AI 티 난다).
- `prefers-reduced-motion` 시 드리프트·전환을 정적/즉시로 대체.

## 핵심 컴포넌트 (라이트/다크 모두 정의)

- **버튼**: primary(악센트 채움)·secondary(라인)·ghost. 라벨은 동작 그대로.
- **필름스트립 클립 카드**: 썸네일 + 순서번호 + 길이(모노). 활성 시 악센트 테두리.
- **자막 인스펙터**: 기본은 텍스트 한 줄. "세부 조정" 펼치면 색·배경·크기·위치.
- **언어 셀렉터 / 테마 토글**: 상단바 우측. 토글은 스위치 형태.
- **빈 상태**: "사진을 올려 시작하세요" + 업로드 영역. 무드 아닌 **행동 유도**.
- **렌더 진행**: 호박색 진행 표시 + 예상 비용/시간.

## 품질 바닥선

반응형(모바일까지), 키보드 포커스 가시화, reduced-motion 존중, 라이트/다크 대비 충족.
악센트는 한 곳에 — "거울 보고 액세서리 하나 빼기."
