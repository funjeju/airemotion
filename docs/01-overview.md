# 01 · OVERVIEW — 제품 정의

## 한 줄 정의

사용자가 가진 **기존 미디어(사진·영상·음악·MP3)를 자동 배열·전환·자막으로 편집**해,
손대지 않아도 완성도 있는 영상을 빠르게 만들어 주는 **편집 자동화 도구**.

## 무엇이 아닌가 (중요)

- ❌ 생성형 영상 AI 아님 — 없던 픽셀을 만들지 않는다.
- ❌ CapCut/프리미어 같은 자유 타임라인 편집기 아님.
- ✅ 사용자가 올린 진짜 소재를, 보기 좋게 **조립·연출**하는 도구.

이 구분이 제품의 핵심 가치다: 결과가 **예측 가능·통제 가능**하고, 원본이 왜곡/환각 없이
그대로 나오며, 비용·속도·저작권 면에서 유리하다.

## 타깃 사용자

전문 편집은 하지 않지만 **그럴듯한 영상이 빠르게 필요한 보편 사용자**.
예: 사진관, 부동산 매물, 행사 요약, 소상공인 상품 소개, 가족 영상.

- **포함:** 안 건드리고 자동으로 끝내고 싶은 사람 + 약간만 다듬고 싶은 사람.
- **비포함(의도적 제외):** 0.1초 단위를 따지는 하이엔드 모션그래퍼. 이들은 전문 도구를 쓴다.

## 핵심 가치 제안

1. **거의 100% 자동** — 리소스 던지고 버튼만 눌러도 완성도 있는 결과.
2. **원하면 딱 그만큼만 개입** — 순서·자막·효과를 단계적으로 조정(progressive disclosure).
3. **일관성으로 완성도 확보** — 효과를 적게, 무난하게 통일해 항상 "괜찮은" 결과.

## 범위 (Scope)

| 포함 | 제외 (Out of Scope) |
|---|---|
| 사진·영상·음악·MP3 업로드 | 픽셀 생성(생성형 AI) |
| 자동 클립 배열 | 자유 드래그 타임라인 편집기 |
| Whisper 자동 자막(검수형) | 하이엔드 모션그래픽 |
| 클립별 자막 입력·스타일 | 영상 내 인물 합성·딥페이크 |
| 켄 번스 애니메이션 + 크로스페이드 전환 | 수십 종 효과·스티커·필터 라이브러리(초기) |
| 구글 로그인 / 다국어 / 다크모드 | 결제·구독(초기 단계 제외, 후속) |

## 성공 기준 (제품 관점)

- 보편 사용자가 **아무것도 안 골라도** 결과를 보고 "이 정도면 충분하다"고 느낀다.
- 리소스 업로드부터 결과 미리보기까지 **막힘 없이** 도달한다.
- 자동 자막은 **초안 → 검수 → 확정** 흐름으로 오타가 그대로 박히지 않는다.
