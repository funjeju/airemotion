# 04 · FEATURES & FLOW — 기능·사용자 흐름

## 핵심 사용자 흐름

```
1. 로그인 (구글)
2. 새 프로젝트 → 리소스 일괄 업로드 (사진·영상·음악·MP3)
3. 대략적 의도 프롬프트 입력  예: "잔잔한 가족 영상"
4. 시스템: 1차 자동 배열 생성 (업로드 순서/시간 기준) → 타임라인 표시
5. 시스템: 음성 소스에서 Whisper 자동 자막 초안 생성 → 타임라인에 표시
6. 사용자: 순서 정렬 + 클립별 자막 입력/수정 + 자동 자막 검수
7. 분기:
     A) "생성" 즉시      → 기본값(자동 효과)으로 출력
     B) 효과·자막 편집 후 → 출력
```

## 기능 명세

### 업로드
- 다중 파일 드래그&드롭 + 파일 선택. 이미지/영상/오디오 혼합 허용.
- Firebase Storage 업로드, 진행률 표시. 메타데이터는 Firestore `clips`.

### 자동 배열
- 업로드 순서(또는 파일 타임스탬프) 기준으로 `order` 부여.
- 사진 기본 노출 4초, 영상은 원본 길이. 오디오는 배경 트랙.

### 자동 효과 배정 (순수 랜덤 금지)
- **사진 애니메이션 풀**: `zoomIn / zoomOut / pan` 중 하나를 **클립마다 번갈아** 배정
  (연속 동일 효과 방지). 무난한 효과만 풀에 있으므로 변주가 섞여도 결과가 안전하다.
- **전환**: 기본 **크로스페이드**로 통일. (옵션으로 slide)
- **프롬프트 톤 매핑**:
  - "잔잔함" → 느린 켄 번스 + 느린 크로스페이드
  - "활기참" → 약간 빠른 전환 + 컷 위주
- 자세한 효과 파라미터는 `docs/05-design-system.md` 모션 섹션과 일치시킨다.

### 자막 — 두 종류
1. **수동 클립 자막**: 각 사진/영상에 사용자가 입력. 전역 기본 스타일 + 클립별 오버라이드.
2. **자동 자막(Whisper)**: 음성 소스에서 추출.
   - **반드시 렌더 전에 생성 → 검수 UI → 확정** 순서.
   - 검수 화면에서 텍스트·타이밍 수정 가능. 수정 시 `source: edited`.
   - 한국어는 다국어 모델, 오인식 가능 → 검수 단계가 핵심 UX의 절반.

### 미리보기
- Phase 1: 브라우저에서 켄 번스 + 크로스페이드 + 자막으로 **재생 미리보기**(서버 렌더 없음).
- 이후 Phase: Remotion 미리보기 ↔ 최종 MP4 렌더.

### 생성/내보내기
- "생성" → (자막 확정 후) 렌더 잡 트리거 → 진행 상태 표시 → 결과 MP4 다운로드/저장.
- 렌더는 워커(Remotion Lambda 등)에서. 비용 추정치 표시.

## i18n (다국어) 동작

- **next-intl** 사용. 기본 `ko`, `en` 제공, 확장 가능 구조.
- URL은 `/[locale]/...`. 상단바에 **언어 전환 셀렉터**.
- 모든 사용자 문구는 `messages/ko.json`, `messages/en.json` 카탈로그에서 온다(하드코딩 금지).
- 최초 로그인 시 브라우저 언어로 초기 `locale` 추정 → `users/{uid}.locale`에 저장, 이후 우선.

### 카피 톤 (디자인 문서와 일치)
- 능동태·평서문. 버튼은 동작을 그대로: "영상 만들기"(Make video), "자막 확정"(Confirm captions).
- 빈 화면은 행동 유도: "사진을 올려 시작하세요." 오류는 무엇이 왜·어떻게 고치는지 안내.

## 다크모드 동작

- **next-themes** (class 전략) + Tailwind `dark:`.
- 상단바에 **토글 스위치**(light / dark). 기본은 `system`.
- 선택값은 `users/{uid}.theme`에 저장해 재방문 시 유지.
- 모든 컴포넌트는 라이트/다크 두 테마에서 검증(05 문서의 두 팔레트 사용).

## 접근성 기준 (quality floor)

- 키보드 포커스 가시화, 모바일까지 반응형, `prefers-reduced-motion` 존중
  (모션 줄이면 켄 번스·전환을 정적 컷/즉시 전환으로 대체).
